# Real-Time Event Synchronization Engine

A full-stack event management platform built with PHP + MySQL.

## Setup
1. Install XAMPP
2. Create database `eventsync` in phpMyAdmin
3. Run `database.sql` to create tables
4. Place `index.php` in `htdocs/project-1/`
5. Visit `http://localhost/project-1/`
